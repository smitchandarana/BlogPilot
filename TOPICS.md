## Relevance Score Thresholds

| Score | Action |
|---|---|
| 0–3 | Skip entirely. Not logged. |
| 4–5 | Log as SKIPPED. No action. |
| 6–7 | LIKE only (unless budget allows comment) |
| 8–9 | LIKE + COMMENT. Visit profile. |
| 10 | LIKE + COMMENT + profile visit + connection if not connected |

Default minimum score to act: **6**
(Configurable in Settings → Feed Engagement → Minimum Score)
